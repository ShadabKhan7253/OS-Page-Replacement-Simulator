# Page Replacement Simulator

## Description
<p>
This app is a simulation to replicate the visualization of different types of page replacement algorithms which includes:
</p>
<ol>
    <li>First In First Out</li>
    <li>Least Recently Used</li>
    <li>Optimal</li>
</ol>
